# Sound Assets

Place your spooky sound files here:

- `thunder.mp3` - Thunder sound for login page transition
- `eerie.mp3` - Eerie background sound for Orkut page
- `haunted.mp3` - Haunted ambience for MySpace
- `retro.mp3` - Retro sound effects for GeoCities

## Recommended Sources for Spooky Sounds:
- Freesound.org
- Zapsplat.com
- Adobe Stock Audio
- YouTube Audio Library

## File Format Recommendations:
- Use MP3 format for web compatibility
- Keep file sizes under 1MB for faster loading
- Use 128kbps quality for good balance of quality/size
