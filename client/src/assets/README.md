# Assets Directory

This directory contains static assets for the Graveyard of the Forgotten Web app.

## Subdirectories:

### backgrounds/
- Spooky background images for different themes
- Fog textures and overlays
- Cemetery and graveyard scenes

### gifs/
- Retro animated GIFs for GeoCities page
- Under construction animations
- Spinning skulls and horror elements
- Classic 90s web animations

### widgets/
- MySpace widget graphics
- Friend box silhouettes
- Pixel art elements

### sounds/
- Thunder and lightning effects
- Eerie ambient sounds
- Retro notification sounds

### bats/
- Bat sprite animations
- Flying creature graphics

### thunder/
- Lightning flash overlays
- Storm effect images

### fog/
- Fog particle textures
- Mist overlay images

## Usage Notes:
- All images should be optimized for web (WebP preferred)
- Keep file sizes reasonable for fast loading
- Use appropriate alt text for accessibility
- Consider dark theme compatibility
