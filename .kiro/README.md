# Kiro Configuration

This folder contains Kiro IDE configuration files for the Graveyard of the Forgotten Web project.

## Structure

- `steering/` - Contains steering rules and project guidelines
- `settings/` - Contains Kiro-specific settings and configurations
- `hooks/` - Contains agent hooks for automated workflows

## Project Overview

The Graveyard of the Forgotten Web is a nostalgic tribute to defunct social media platforms, featuring:
- Authentic recreations of MySpace, Orkut, and GeoCities
- Interactive vampire chatbot with Count Dracula
- Gothic horror atmosphere with atmospheric effects
- Full-stack implementation with React, Node.js, and MongoDB
