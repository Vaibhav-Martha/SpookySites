# Project Guidelines - Graveyard of the Forgotten Web

## Project Overview
This is a nostalgic tribute to defunct social media platforms with a gothic horror theme.

## Architecture
- **Frontend**: React with Vite, TailwindCSS, GSAP animations
- **Backend**: Node.js, Express, MongoDB with Mongoose
- **Authentication**: JWT-based with bcrypt password hashing
- **Theme**: Gothic horror with vampire elements

## Code Standards
- Use functional React components with hooks
- Implement responsive design with TailwindCSS
- Follow RESTful API conventions
- Use proper error handling and validation
- Maintain consistent gothic/horror theming

## Platform Recreations
- **MySpace**: Blue theme, Top 8 friends, custom HTML/CSS
- **Orkut**: Pink/purple theme, scraps, testimonials
- **GeoCities**: Retro web design, under construction elements

## Vampire Chat Features
- Royal gothic fonts (Cinzel, Playfair Display)
- Dark red/black color scheme
- Sophisticated AI responses in character
- Medieval castle atmosphere

## Development Workflow
- Test all features before deployment
- Maintain authentic platform designs
- Ensure responsive layouts work on all devices
- Keep gothic atmosphere consistent throughout
